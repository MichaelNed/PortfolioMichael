RESPONSIVE
YOUTH CARE DASHBOARD

In partnership with the municipality of Oosterhout, I have made a responsive dashboard combined with machine learning to visualize & analyze their youth care dataset and to help them improve their business processes.

Pre-processing
A PostgreSQL database is used to prepare the used dataset. The data is cleaned and checked by using various SQL queries. The database is connected with the visualization tool to create a real-time connection with the data. This gives reliable and up-to-date insights into youth care processes.
Analysis & Visualization
Using Microsoft PowerBI, I developed interactive visualizations of the data in a dashboard. I have applied the organization's color palette used in their branding (orange & dark blue). To make the dashboard more accessible for everyone in the organization, I added an information button in the upper left corner of each page.
Machine Learning
I created a machine learning algorithm that predicts if a future client is likely to quit the care early. This algorithm is also implemented in the dashboard. A showcase of this page is not possible due to privacy concerns. The municipality can use this algorithm to give the clients the best care that fits the best.
Ethics & Law
Healthcare-related data is highly sensitive. The data is completely anonymized and therefore no individual can be traced back. The data falls under the AVG (GDPR), and many other data privacy laws as listed on the municipality's website. In addition, a Data Ethics Decision Aid (DEDA) was utilized when developing the machine learning algorithm.
